# ProxiChat
Geolocation Based Chat App with React Typescript and Firebase

## Cloning branch
```shell
git clone https://github.com/NounceChat/Nounce.git
```

### Run
```shell
cd Nounce
```
```shell
docker compose up
# or 
npm i
npm run dev
```
